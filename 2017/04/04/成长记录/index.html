April 4, 2017

成长记录

2017-04-11

今天笑来老师的一句话，把我感动哭了，原话是这样的

事实上，在这里留言，是写给自己看的，写给未来某一天的你看的。终究有一天，你会「突然想起来」回头看看自己写过什么~ 然后，因为你一直在写，一直在思考，于是一直在进步。于是，已经进步了的未来的你，看到今天的你曾经写过的东西，一方面会乐：「我那时候真是一点水平也没有啊！」另一方面会很开心：「那我现在进步还是很大的嘛！」

当我看到「已经进步了的未来的你，看到今天的你曾经写过的东西」时，我想到的是，「我会感激曾经努力的自己，没有那时的自己，就没有现在的我！」想一遍哭一遍，也不知道为什么会有这么大的感染力，这种穿梭时空的对话，非常奇妙，这种感恩是发自肺腑的，不只是未来的我才会有，就像今天的我感恩昨天努力的自己一样，只是时间维度被拉长了许多，我猜大概是这种使命感把今天的我感动得一塌糊涂吧！

今天把《好好学习》的最后一章读完了，在书本的结尾，成甲感谢了非常多的人，哪个人给予了他什么帮助，他都记得一清二楚，虽然他感谢的不是我，但我眼泪还是不自觉得流了出来，因为我感受到这种发自肺腑的感恩，真诚、虚心。不过我觉得他还少感恩了一个人，那就是熬夜写书的自己。感恩是一个优秀的品质，然而自己却没拥有，不说别的，就连自己的家人也很少想着去感恩吧，其实父母并没那么需要我给他们的物质，毕竟自己也知道，钱买不到真快乐，他们更需要的是爱，出来这么久了，也没有给电话问候，虽然他们已经习惯了我不在身边的日子，但是起码给个电话让他们知道，我在想念他们，那他们心里也会暖暖的，我从小就不像我姐善于表达，能给到父母的爱也少之又少，估计他们也习惯了，现在长大了，该补回了吧！说完该行动了。

每次看到笑友在文章下面的评论，我的挫败感就回来了，为什么不管在什么地方，什么领域，总会有一大波人成长得比自己快那么多，以至于被差距吓到都寸步难行，快要放弃了。幸运的是，我又发现不只是我一个人有这样的感受，不少人都有，那说明我在队伍里也不至于太落后，于是又有勇气继续摸爬滚打了。可是回到生活中，我又有点不适应了，因为反差太大，身边在努力成长的人屈指可数，难道他们都没有意识到知识大爆炸的时代来临了吗，不学习不成长，将很快就会被社会淘汰，认知将是一个人能力的重要体现。我真希望身边的人都一起成长起来，那样我沉浸在这个环境中也会更有动力去成长，但是，身微言轻吧，只好让自己成长得足够成为他们的榜样再去影响他们了。

2017-04-10

发现自己对文字开始喜欢了，不管是输出还是输入，缺一不可。最近的几个月，每天都有大量的文字输入，看问题虽然还不够深入，但是也没之前那样肤浅，面对陌生的知识也开始有耐心深入探索了，没有之前那样浮躁。说到这，就顺便总结一下自己这几个月的变化，量化自己的成长，给自己一些正反馈，让自己更有动力前进。

  1. 更舍得花钱了

    最主要的表现是付费阅读，之前遇到付费还会犹豫不决，现在遇到了自己喜欢的，想要的内容，就会果断买下。金钱对现在的我来说就像死灰一样，没有生机，不会增值，我根本不知道怎么用这些金钱去创造更多的价值，如果攒着，这点钱就算攒个十年，也没多少，还没笑来老师一个讲座赚的多，所以也没必要攒着。现在对我来说最划算的是用这些钱让自己快速成长起来，只有自己成长起来了，才有能力去创造更大的价值，这样做是充分利用自身的复利效应，增长率最大化。所以遇到能补充自己的知识，提高自己的效率，节约自己的时间的事情要舍得花钱。

  2. 更在意效率了

    效率是个好东西，提高效率，能帮助我们在同样的时间做更多的事情，或者吸收更多的内容，这就相当于延长了我们的生命呢。要有意识得提醒自己注意效率，有时候时间一充足，自己就会变得懒散，效率降低。之所以会觉得时间充足，是没有意识学习是没有止境的，我们不可能有多余的时间让我们偷懒。

  3. 学会坚持了

    其实说的不够准确，我以前也是有坚持的事情的，只不过是坚持「打游戏」，现在是坚持「阅读」、「健身」。之所以能坚持做，一定是这件事情起到了一定的正反馈，「打游戏」之所以能让我「坚持」，因为游戏拼的是技术，当自己技术达到一定高度时能在游戏里「秀」别人，从而产生了一种「优越感」，而这种感觉在现实生活中是十分难获得的，所以才能「坚持」。而「阅读」、「健身」同样是对我自身产生了改变，「阅读」让我视野更开阔，让我用截然不同的眼光看世界，「健身」带给我的变化很明显，手臂肌肉结实了，线条明显了。这些改变都会让人上瘾，从而产生了复利效应，越做效果越明显，效果明显了就越想做。

  4. 执行力更强了

    这个跟效率也是有关系的，为了提高效率，执行力必须提高。而且我也深信，光思考是没有卵用的，思考后要有所选择，选择了要有所行动，只有行动了才是真正得改变，而且行动了才能有所反馈，矫正或者印证我们的想法。

  5. 更理性了

    其实我一向都很理性，现在理性得更有依据了，喜欢一本正经得讲道理，但是由于度把握不好，导致过于严肃，幽默感下降。

  6. 消极的想法大大减少了

    因为我知道，很多事情起步晚了，后悔也没用，最好的补救方法就是从现在开始行动，让损失降到最低，而且其实自己还年轻，今年才23岁，我也能看到，不到半年时间，我的改变就这么大，我相信，只要我的认知能力上去的，我的成长会越来越快的，所以我想订个小目标，两年后的今天，我要对经济学、管理学、心理学这三个领域都有一个系统的认知。

  7. 生活更有秩序了

    最新特别习惯去使用记事本这样的东西，把当前的工作任务、工作中遇到需要学习的知识点、碎片知识点都记下来，只要一闲下来就看看本子上的任务或者知识还有哪些没有完成或者消化的，让自己效率更高。

2017-04-09

刚刚突然发现，已经两天没写日记了，于是挺好奇我这两天晚上到底在干嘛，昨天我记得是看「得到」的直播看得挺晚的，很精彩，喜欢罗永浩的真诚，还有他的锤子的设计理念，注重用户体验并且增加了「Big Bang」和「One Step」这样的提高效率的工具，如果不是我 iPhone5s 还没坏，我估计就换了，毕竟苹果最近也没啥创新。那前天干嘛去了呢？实在想不起来了，还好有用「ATimeLogger」的进阶版「timetrack」，于是打开手机查看了周五晚的活动，这才想起来，原来我那天晚上在学 Java 的「反射原理」，后来又上了会网，好吧。

今天搭地铁给「燕姐」送光盘的时候，发现地铁实在是太吵杂了，而且人流穿梭，十分影响看书的专注度，进而影响了效率，虽然最近看书一直专注不了，还是要不断努力去调整，尝试着改变现状，于是我觉得闭着眼睛戴着耳机听书应该是个不错的选择，然后打开「得到」花了 ￥4.99 买了「成甲说书：自控力」。果然，效果不错，效率有提高一些，但是光听的话总觉得好像欠缺了什么，得打开备忘录稍微记一下效果会更好，而且得听多几遍，每遍都会听到新的细节，才发现原来上一遍在这个地方走神了没听到，或者没用心思考过，左耳进右耳出了，听到最后，能把整个内容罗列出来，这才算听完整了。我罗列一下今天「自控力」的大纲。

  1. 作者跟我们阐述了意志力的特性，它会随着自身的状态不断变化的，是会被消耗和恢复的，人不能一直保持较高的意志力。所以要接受意志力低下时候的自己，同时也要学会原谅这样的自己。
  2. 生活中容易导致意志力低下的坑
    • 取得进步后的放松
    • 觉得自己明天对事情的接受程度会比今天高得多
    • 被行为的表象结果所欺骗（刷微博使我快乐）
  • 抵挡不住眼前的收获，出售自己的未来
    • 破罐子破摔
    • 近朱者赤近墨者黑
  1. 提高意志力的几个方法
    • 冥想
    • 深呼吸

由于下午有整理过内容，所以记得还算比较清晰的。光记住是没用的，接下来得花苦功夫在践行上，首先得辨别出这几个坑，避免自己掉进去，然后还要利用元认知寻找更多自己并没无意识发现的坑。

我觉得自己平时看公众号积攒的碎片知识太多了，看过了脑袋只有一个印象，十分零散，很难有一个整体的概念整合到操作系统中去，但是也不可否认碎片知识的价值，我觉得还是得靠「记」，就是看完后能用自己的话把关键知识点阐述出来，能够关联起自己的旧知识，然后在手机或者电脑里记下来，这样才能更有效的一点一点修补操作系统，而整体的概念的话还是得多看书。

2017-04-06

讲真，今晚我好想哭，为时光匆匆流逝而感慨，为美好岁月一去不复返而惋惜。人啊，就不该老往后看，活在过去容易长不大，但是吧，有时候回头看看，其实也会有意想不到的收获，看着照片，熟悉又陌生的感觉，会有一些特别触动的瞬间，想去珍惜却已经来不及，转而想着珍惜当下身边的人和事，因为知道，这些瞬间也都将成为过去，值得回味。

那年，我度过了那时最长并且最悠闲没有暑假作业的暑假，那是中考后的暑假。我们十二人去了欢乐谷，六男六女，现在想想，这应该是最后一次男女比例如此均衡的活动的。我是第一次跟这么多同学出去玩，而且都是特别 friend 的，玩的特别开心，让我最难忘的是玩「激流勇进」，没穿雨衣的那种，结果我们都意料之中得湿身了，年轻就是疯狂，哈哈最佩服的还是那几个女生，后来我们在太阳底下晒了半小时才晒干。看着照片里熟悉的模样，单纯，青涩，傻不拉几，然后想想他们现在的样子，变化大的让我吃惊，女生都变得成熟漂亮了，也会穿衣打扮了，男生呢，说来也是惭愧，很久没联系了，也不知道发展得怎么样了，这点让我挺心痛的，当时不懂事，不知道友谊的珍贵，没有去刻意维系这份友谊，觉得顺其自然就好，友谊会长存，却不知道，涌入各自生活的人会越来越多，不花时间去维系的友谊只会渐渐淡去，最终被替代。

还有高一那段时光也让我挺舍不得的，高一11班是一个优秀的群体，因为优秀，所以更懂得包容，让我感觉每个人都可爱，相处的很舒服。而后因为要分文理科，也因为我成绩差，所以就这样跟这班人分开了，后来也越来越少联系了。

初三、高一距离现在，差不多七八年时间吧，按李笑来的话，七年就是一辈子，现在看来，这一辈子确实让不少人产生翻天覆地的变化。我觉得自己这七年，跟他们比起来，过的非常差劲，但是不要紧，我对自己有信心，现在只是这个七年的第一年。

总结

  1. 朋友是需要花时间去维系的。
  2. 七年很长，对一个人的改变可以很大，也可以很小。
  3. 珍惜身边的人，最好能用相片记录曾经共同的瞬间。感恩。

2017-04-05

感觉自己说话还是太严肃、生硬，容易成为话题终结者。仍然处于自我防卫的状态，对外界刺激过于敏感，随时准备着反击。

做事情依然没有讲究战略，一想到就一股脑的去做，就像今天，突发奇想想去国家图书馆看看，于是提前到五点下班就去了，去到已经五点半了，才知道已经闭馆了，拿出手机一查发现五点就闭馆了，这样就白白浪费了很多时间在交通上，导致效率低下，为什么就不在想到去的时候就了解清楚状况，做好充分准备呢？做事还是要有脑子比较好吧，学了要用起来才能算学到，学了不用就是白学！

2017-04-04

昨天晚上焦虑得久久不能入睡，因为两件事，都跟找工作相关的。

第一件事，是我在「小密圈」里看到一个小伙伴发的求职经历，他跟我一样大，211学校毕业，大学时候担当电竞协会会长，打过比赛，办过活动，一直有着一个电竞梦。他上一份工作是一个银行外包的开发，也是得在外头出差，一待就待好几个月，我觉得他的技术比我好得多，因为他列举的一些面试题我一点概念都没有，而他能很快得做出来。这样的一个在我看来比较优秀的同龄人，竟然也找了三十三天才找到工作。

第二件事，是看到另外一位小伙伴博客上面置顶的一篇简历。那是他的简历，92年生，比我大一岁，专升本毕业，已经有三年工作经验，现在是上海某公司里 Android 项目负责人。在他的简历下面，密密麻麻得列举了他所接触的技术和项目，我不由自主得拿自己和他做对比，技术知识储备粮少了别人一大截，项目也没别人多，而且，现在我做的项目也真的拿不出收，没有一点技术含量。

这两件事让我产生了巨大的挫败感，一丁点跳槽的勇气都没有了，更别说创业了。为什么大学同样是混日子，差距竟然会如此大，起码他大学还管理过社团，举办过活动，领导力、人脉多多少少也是有提升的，而我连社团都没参加过，除了打游戏、谈恋爱，宝贵的大学四年时光好像真的被我过的分文不值。想到这里我真的难受！心痛！眼泪也流了下来。可是这一切都已成定局，后悔又有什么用呢。四年带来的差距还是蛮大的，想通过几个月的时间就完全消灭差距的可能性实在太低，我现在要做的是要平复情绪，客观冷静思考，然后想办法尽可能的缩小差距，我觉得我得比别人更努力才行，不仅每天的学习时间要更长，效率也要更高，毕竟我还年轻，才二十三岁，觉悟的还不算太晚，给自己两年时间，我觉得这差距一定可以完全消除！为了便于落实行动，我要具体列举一下「努力」的计划。

  1. 早上七点起床，看「得到」、「好好学习」、罗列当天任务等。
  2. 管理好自己的注意力，每天只给自己15分钟刷朋友圈。
  3. 做任何事情都要注重「效率」，有一定的战略思考。
